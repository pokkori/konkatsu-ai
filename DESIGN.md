## 4. 機能仕様

### プロフィール添削
- 入力：アプリ・年齢・職業・趣味・現プロフィール文・改善ポイント
- 出力：改善版プロフィール文 + 改善ポイント解説(3〜5点) + 追加アドバイス

### メッセージ文案生成
- 入力：相手プロフィール・メッセージ目的・自分のキャラクター
- 出力：3パターンのメッセージ案（パターン名・本文・効果的な理由）

### 返信分析
- 入力：相手からの返信テキスト
- 出力：脈あり度(%)・感情分析・次のメッセージアドバイス

## 5. API仕様

### POST /api/profile
**リクエスト**
```json
{
  "app": "Pairs",
  "age": "28",
  "job": "ITエンジニア",
  "hobbies": "カフェ巡り、映画",
  "profile": "はじめまして...",
  "goals": ["appeal", "match"]
}
```

### POST /api/message
**リクエスト**
```json
{
  "targetProfile": "カフェが好きで...",
  "purpose": "最初のメッセージ",
  "character": "明るい"
}
```

### POST /api/analyze
**リクエスト**
```json
{
  "reply": "そうなんですね！私もカフェ好きです"
}
```

### POST /api/create-checkout
**リクエスト**
```json
{ "plan": "popular" }
```
